`Badge` — etiqueta compacta tonal (fundo suave, texto saturado). Não use cores chapadas.

```jsx
<Badge variant="primary">Novo</Badge>
<Badge variant="outline">Rascunho</Badge>
```

Variantes: `neutral` · `primary` · `clay` · `success` · `warning` · `destructive` · `outline`. Para status de negócio (pago/pendente/atrasado) use `StatusBadge`.
